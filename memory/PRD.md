# GlowWithLucy - Candle Business Website

## Original Problem Statement
Make a website for GlowWithLucy a bespoke, handmade candle business. Beige website with 7, 9 and 12 oz candle sizes. Add "Built by DH Website Services" to the footer. Multi-page site with contact-to-order approach.

## User Personas
- **Lucy** - Business owner who makes handmade candles and needs to showcase products
- **Customers** - People looking for bespoke, handcrafted candles with custom scents

## Core Requirements
- Multi-page website (Home, Shop, About, Contact)
- Beige/cream color scheme matching logo
- 3 candle sizes: 7oz, 9oz, 12oz
- Contact-to-order approach (no shopping cart)
- Contact form with scent preference field
- Display email: lucyd789@sky.com
- Footer credit: "Built by DH Website Services"

## What's Been Implemented (Dec 2025)
- [x] Homepage with hero section, featured candles, values, CTA
- [x] Shop page with 3 candle sizes and "Enquire Now" buttons
- [x] About page with story, values, and process sections
- [x] Contact page with form (Name, Email, Size dropdown, Scent Preferences, Message)
- [x] Responsive navigation with mobile menu
- [x] Footer with quick links and "Built by DH Website Services"
- [x] Backend API for contact form submissions (POST /api/contact)
- [x] Beige/cream design theme with Playfair Display + Outfit fonts

## Tech Stack
- Frontend: React with Tailwind CSS, Shadcn UI components
- Backend: FastAPI with MongoDB
- Fonts: Playfair Display (headings), Outfit (body)
